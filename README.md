## Byte Menu

O Byte Menu é um sistema desenvolvido para atender às necessidades da cantina do SESI/SENAI.

A proposta do sistema é simples: permitir que você compre seus produtos antes de chegar à cantina, evitando filas e agilizando o atendimento.

Assim que o seu pedido estiver pronto, basta ir até o local indicado e realizar a retirada.
